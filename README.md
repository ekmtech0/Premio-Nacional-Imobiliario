# Prémio Nacional Imobiliário

**Prémio Nacional Imobiliário 2025 — celebrando a excelência em arquitetura, serviço público, mediação e sustentabilidade.**

## Descrição

Este repositório contém a plataforma oficial de votação online para o Prémio Nacional Imobiliário de Angola. O objetivo é reconhecer e premiar a excelência no setor imobiliário angolano, destacando projetos de arquitetura, serviços públicos de destaque, iniciativas de mediação e esforços de sustentabilidade.

## Funcionalidades

- **Votação Online:** Permite aos utilizadores votar nos melhores projetos e profissionais do setor imobiliário de Angola.
- **Gestão de Categorias:** Suporta diversas categorias, como arquitetura, serviço público, mediação e sustentabilidade.
- **Resultados em Tempo Real:** Apresentação dos resultados de votação de forma transparente e em tempo real.
- **Administração Segura:** Interface de gestão para administradores configurarem categorias, candidatos e acompanharem as votações.

## Tecnologias Utilizadas

- **Frontend:** Vue.js (57.5%)
- **Backend:** C# (38.3%)
- **Scripts auxiliares:** JavaScript (3.3%)
- **Outros:** (0.9%)

## Instalação

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/ekmtech0/Premio-Nacional-Imobiliario.git
   ```
2. **Acesse o diretório do projeto:**
   ```bash
   cd Premio-Nacional-Imobiliario
   ```

3. **Configure e instale as dependências do frontend (Vue.js):**
   ```bash
   cd frontend
   pnpm install
   pnpm run dev
   ```

4. **Configure o backend (C#):**
   - Instale as dependências via .NET CLI:
     ```bash
     cd ../backend
     dotnet restore
     dotnet run
     ```

   > Siga as instruções específicas em cada subdiretório se existirem.
